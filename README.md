# CMMComplier
CMM语言解释器
## 词法分析
### 词法分析主要功能
a)忽略空格、tab键、回车换行等分隔符；<br>
b)识别不同类型的记号；<br>
c)识别并忽略注释；<br>
d)记录下每个记号的行号与列号；<br>
e)将识别的记号输出；<br>
f)如果输入串存在词法分析错误，则报错。<br>
